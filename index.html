Deploy me on Vercel

Simple Vercel Demo

This is the smallest useful website for explaining a GitHub-to-Vercel deployment. It is just one file: index.html.

What you need

  • A GitHub account.
  • A Vercel account connected to GitHub.
  • A repository with this index.html file in the root.

How deployment works

  1. Push this folder to a GitHub repository.
  2. In Vercel, choose New Project.
  3. Import the GitHub repository.
  4. Use the default settings and deploy.

No framework, no install command, no build command, and no environment variables are needed for this demo.
